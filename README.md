# Acceso a Datos - 2º DAM

Repositorio dedicado a los ejercicios y prácticas de la asignatura **Acceso a Datos** del segundo curso del Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Multiplataforma (DAM).

## Descripción

Este repositorio contiene el material práctico desarrollado durante el curso, incluyendo ejercicios, proyectos y ejemplos de código relacionados con:

- Gestión de ficheros
- Bases de datos relacionales
- Bases de datos NoSQL
- Frameworks de mapeo objeto-relacional (ORM)
- Acceso a datos mediante conectores
- XML y JSON
- Programación de componentes de acceso a datos

## Uso

Cada carpeta contiene ejercicios específicos con sus propias instrucciones de ejecución.
